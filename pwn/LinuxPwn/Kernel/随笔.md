#### 调试内核时断点没有正确下达

下的断点时没有加内核模块基址的偏移，需要指定每个段的信息

```
add-symbol-file mydrivers/ko_test.ko  0xffffffffc0002000 \
                    -s .rodata.str1.1 0xffffffffc000304c \
                    -s .symtab        0xffffffffc0007000 \
                    -s .text.unlikely 0xffffffffc0002000
```

之后断点正确加上了偏移
