#### firmadyne

这个软件貌似较难安装，直接使用attifyti提供的物联网渗透专用虚拟机，上边已经安装好了绝大部分分析固件的工具，包括firmadyne

下载地址：[https://www.dropbox.com/sh/xrfzyp1ex2uii53/AAAF0mdA1qFaEBDYZoIxaQRma?dl=0](https://www.dropbox.com/sh/xrfzyp1ex2uii53/AAAF0mdA1qFaEBDYZoIxaQRma?dl=0)

user: `oit`

password: `password@123`

##### 使用

`python fat.py` 运行firmadyne

输入固件所在地址，输入固件类型，遇到 `Password for user firmadyne:` ，输入数据库密码 firmadyne

之后若成功运行则显示IP

#### FirmAE

安装

```
git clone --recursive https://github.com/pr0v3rbs/FirmAE
cd FirmAE
./download.sh
./install.sh
```

使用

```
sudo ./run.sh -a dlink /home/i/iot/dlink/DIR-868L/DIR-868L_fw_revB_2-05b02_eu_multi_20161117.zip
```

默认的模拟地址可能和网关冲突，需要修改一下
