#### mi_wifi中lua反编译

小米路由ax3600、ax1800解包后发现lua脚本加密，使用unluac进行反汇编

```
git clone https://github.com/NyaMisty/unluac_miwifi.git
cd unluac_miwifi
mkdir build
javac -d build -sourcepath src  src/unluac/*.java
jar -cfm build/unluac.jar src/META-INF/MANIFEST.MF -C build  .
```

使用

```
java -jar /home/i/unluac_miwifi/build/unluac.jar rpc.lua > rpc.o.lua
```

#### 修改系统ASLR

```
sudo sysctl -w kernel.randomize_va_space=0	# 临时修改，重启后恢复
cat /proc/sys/kernel/randomize_va_space  	# 验证修改结果
sudo sysctl -w kernel.randomize_va_space=2 	# 调试完成后恢复默认
```

#### dd命令

用于从指定的固件文件中提取特定范围的数据并保存为新文件

```
dd if=wdr7660gv1-cn-up_2019-08-30_10.37.02.bin of=uboot.raw bs=1 skip=512 count=66048

参数：
if：input file，指定要读取数据的源文件
of：output file，指定输出文件
bs：block size，设置每次 读取/写入 的字节数
skip： 跳过输入文件的前x字节，从第x字节开始读取
count：选择读取的字节数

```
